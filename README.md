# Placement & Internship Tracking System

A backend-based web application built using Python and Flask that helps students track internship and placement applications efficiently.

## Features
- User Registration & Login
- Session-based Authentication
- Add and track company applications
- Track application status and deadlines
- User-specific dashboard
- Secure route access
- Clean and simple UI

## Tech Stack
- Backend: Python, Flask
- Database: SQLite
- Frontend: HTML, CSS
- Tools: VS Code

## Project Structure
- app.py – Main backend logic and routes
- templates/ – HTML templates
- static/ – CSS styling
- database.db – SQLite database

## How It Works
1. Users register and login
2. Sessions are created after authentication
3. Logged-in users can add applications
4. Dashboard displays user-specific data
5. Logout clears session
